## 🌐 3D Web Developer Portfolio

A fully interactive, animated 3D portfolio website built using **React**, **Three.js**, **@react-three/fiber**, and **GSAP**.  
This portfolio showcases development skills, projects, and contact information with a modern design and immersive visuals.

### 🚀 Technologies Used

- **React** – Component-based UI development
- **Three.js** & **@react-three/fiber** – 3D scenes and animations
- **GSAP** – Smooth, scroll-triggered animations
- **Vite** – Fast dev server and build tool
- **Netlify** – Continuous deployment & hosting

---

## ✨ Features

- 🌟 Scroll-triggered GSAP animations
- 🧠 SplitText for detailed text reveals
- 📦 Modular component structure
- 🖥️ Interactive 3D scene powered by Three.js
- 📱 Fully responsive across all screen sizes
